Japanese Flash Cards App

This is a Japanese flash card app built using Tkinter GUI library in python, the user will guess the meaning of the work in japanese and if he knows the meaning he can top correct button or if not he can tap no button, If the user doesn't do anything for 5 seconds the meaning of teh word will be revealed.

Vamz Password Manager

This is a simple password manager built using Tkinter GUI library along with other libraries such as random, Pillow, etc. The user can generate a secure random password for a website and also store it inside the password manager, he can also search for the password using the search button.

Space Invaders tkinter game

This is a remake of the old classic retro game - Space Invaders, thee user will have to defend & destroy the invader's fleet. The game is built using the Turtle libray in python and also have sound effects in the game.

PDF to text or Mp3 app

This is app which can extract the text of the pdf and additionally it can also generate the audiobook of the pdf, its interface has been built using tkinter library, and for its functionality I have used modules such as PYPDF2 and google speech api.,

Image Watermarking App

This is an simple Watermarking app which is built only using tkinter and pillow module, user can load an image into the program and then input the text they desire, then the output image will be extracted with watermark.

Vonnie's Arrival Website

This is an dummy online shopping website I have built for my Girlfriend,I have used using flask framework and Bootstrap with Jinja templating, it displays sections of the fashion items on the shop and user can check the item and proceed to buy them.
